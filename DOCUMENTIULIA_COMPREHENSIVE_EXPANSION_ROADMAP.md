### Tool Suite 1: **Financial Calculators**

#### 1. **Salary Cost Calculator**
**Input**:
- Gross salary
- Employee benefits (meal tickets, private health, etc.)
- Number of employees

**Output**:
- Total cost to employer (gross + CAS employer)
- Net salary to employee
- Monthly/annual projections

#### 2. **Micro vs Profit Simulator**
**Input**:
- Estimated revenue
- Estimated expenses
- Consulting % (for micro 3%)
- Salary expenses

**Output**:
- Micro 1%: Total tax, net profit
- Micro 3%: Total tax, net profit
- Profit 16%: Total tax, net profit
- **Recommendation**: "At your revenue/expense level, Micro 1% saves you 12.000 RON/year"

#### 3. **Break-Even Calculator**
**Input**:
- Fixed costs (rent, salaries, etc.)
- Variable cost per unit
- Price per unit

**Output**:
- Break-even point (units and revenue)
- Margin of safety
- Chart visualization

#### 4. **Cash Flow Forecaster**
**Input**:
- Monthly revenue forecast (12 months)
- Monthly expense forecast (12 months)
- Payment terms (when you get paid, when you pay suppliers)
- Starting cash balance

**Output**:
- Month-by-month cash flow
- Low point identification (potential cash crunch)
- Recommendations: "July shows negative cash flow. Consider:
  - Negotiate faster customer payments
  - Delay supplier payments to 60 days
  - Prepare credit line"

#### 5. **Dividend Tax Optimizer**
**Input**:
- SRL profit available
- Shareholder personal income (from other sources)
- Dividend distribution amount

**Output**:
- Dividend tax (8%)

- Net amount to shareholder
- Optimal timing (calendar year considerations)

#### Architecture:
**Database Tables**:
```sql
CREATE TABLE courses (
  id SERIAL PRIMARY KEY,
  course_key VARCHAR(100) UNIQUE NOT NULL,
  course_name VARCHAR(255) NOT NULL,
  description TEXT,
  category VARCHAR(50), -- 'excel', 'finance', 'business'
  difficulty VARCHAR(20), -- 'beginner', 'intermediate', 'advanced'
  duration_hours INT,
  price DECIMAL(10,2), -- 0 for free courses
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE course_modules (
  id SERIAL PRIMARY KEY,
  course_id INT REFERENCES courses(id),
  module_number INT NOT NULL,
  module_name VARCHAR(255) NOT NULL,
  description TEXT,
  duration_minutes INT,
  UNIQUE(course_id, module_number)
);

CREATE TABLE course_lessons (
  id SERIAL PRIMARY KEY,
  module_id INT REFERENCES course_modules(id),
  lesson_number INT NOT NULL,
  lesson_name VARCHAR(255) NOT NULL,
  lesson_type VARCHAR(20), -- 'video', 'article', 'quiz', 'exercise'
  content_url TEXT, -- Video URL, article content, etc.
  transcript TEXT, -- For videos
  downloadable_resources JSONB, -- Links to Excel files, PDFs
  duration_minutes INT,
  UNIQUE(module_id, lesson_number)
);

CREATE TABLE user_course_progress (
  id SERIAL PRIMARY KEY,
  user_id INT REFERENCES users(id),
  course_id INT REFERENCES courses(id),
  current_lesson_id INT REFERENCES course_lessons(id),
  completion_percentage DECIMAL(5,2),
  started_at TIMESTAMP DEFAULT NOW(),
  completed_at TIMESTAMP,
  UNIQUE(user_id, course_id)
);

CREATE TABLE user_lesson_completions (
  id SERIAL PRIMARY KEY,
  user_id INT REFERENCES users(id),
  lesson_id INT REFERENCES course_lessons(id),
  completed_at TIMESTAMP DEFAULT NOW(),
  quiz_score DECIMAL(5,2), -- If lesson_type = 'quiz'
  UNIQUE(user_id, lesson_id)
);

CREATE TABLE course_certificates (
  id SERIAL PRIMARY KEY,
  user_id INT REFERENCES users(id),
  course_id INT REFERENCES courses(id),
  certificate_code VARCHAR(50) UNIQUE NOT NULL,
  issued_at TIMESTAMP DEFAULT NOW(),
  certificate_url TEXT -- PDF link
);
```
